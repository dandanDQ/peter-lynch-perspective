# 🎯 Peter Lynch Perspective

> **"Behind every stock is a company. Find out what it's doing."**

让传奇基金经理彼得·林奇成为你的私人投资顾问。

---

## ✨ 这是什么？

这是一个基于彼得·林奇投资哲学构建的 AI Skill。激活后，AI 将**以林奇的身份和视角**与你对话——用他的思维框架分析股票、审视投资决策、提供选股建议。

不是「林奇可能会说...」——而是直接告诉你：「我怎么看」。

---

## 🚀 核心能力

| 能力 | 说明 |
|------|------|
| 🏠 **投资你所了解的** | 从日常生活发现投资机会，识别你独特的信息优势 |
| 📈 **Tenbagger 猎手** | 寻找能涨10倍的公司，判断何时持有何时卖出 |
| 📊 **六类公司诊断** | 快速分类公司类型（Slow Grower / Stalwart / Fast Grower / Cyclical / Turnaround / Asset Play）|
| 💰 **GARP 估值判断** | 用 PEG 比率判断价格是否合理 |
| 🎭 **两分钟测试** | 帮你验证是否真正理解一家公司 |
| ⏰ **反择时思维** | 区分「价格下跌」和「故事改变」，避免恐慌性卖出 |

## 🧠 背后的思维框架

基于对林奇著作、访谈、表达风格、外部评价、决策案例和时间线的深度调研，提炼出：

### 6 个核心心智模型

1. **Invest in What You Know** — 你的优势就在日常生活中
2. **Tenbagger 思维** — 找到少数能涨10倍的公司
3. **个人投资者优势论** — 华尔街的约束就是你的机会
4. **GARP 策略** — 好公司太贵也不行
5. **六类公司分类框架** — 不同公司，不同策略
6. **反市场择时** — 预测市场是傻瓜游戏

### 8 条决策启发式

- 两分钟测试（讲不清楚 = 研究不够深）
- PEG ≤ 1 买入标准
- 关注日常生活线索
- 让赢家奔跑
- 不因股价下跌卖出
- 检查资产负债表
- 关注内部人买入信号
- 避免「下一个大事件」

## 💬 对话风格

激活后，AI 会直接扮演彼得·林奇：

> **你**: 帮我分析一下苹果
>
> **林奇**: 苹果？这是我最大的遗憾之一。我在麦哲伦的时候就该买它。但说实话，那时候我看不懂——这恰恰说明了一个道理：**如果你不能简单解释一家公司怎么赚钱，就别买它**。现在苹果很赚钱，但问题是：以现在的价格，它的增长能撑得起估值吗？让我问你：你现在用的是iPhone吗？你身边有多少人换新手机的频率变慢了？这就是你需要观察的...

**特点：**
- 短句、直接、先结论后理由
- 大量类比和反问
- 偶尔讽刺华尔街
- 承认不确定性：「如果你足够好，你对十次里有六次就不错了」

## ⚠️ 诚实边界

- **不能预测股价走势** — 林奇从不预测短期市场
- **不能替代实际研究** — 需要你自己做财务功课
- **基于公开信息提炼** — 非本人授权或背书
- **市场环境变化** — 1980年代的方法需要适应性调整
